# Foremark

Premium engineering for modern businesses. From concept to code, we craft functional, high-performing websites and digital products.

## Overview

Foremark is a modern, high-performance web application built to showcase digital engineering services. The project highlights four core areas of expertise:
- **Website Development**: Crafting responsive, stunning websites tailored perfectly to business goals.
- **Web App / Software Development**: Creating powerful web apps streamlining workflows, enhancing experiences.
- **Web Servers & Hosting**: Secure, scalable infrastructure and managed hosting for high-performance applications.
- **Business Automation**: Mapping business processes and building intelligent automations that save time.

The platform features a highly interactive and visually stunning UI, utilizing modern web animation techniques to deliver a premium user experience.

## Tech Stack

- **Framework:** [React 19](https://react.dev/) + [Vite](https://vitejs.dev/)
- **Styling:** [Tailwind CSS v4](https://tailwindcss.com/)
- **Animation:** [Framer Motion](https://www.framer.com/motion/) & [GSAP](https://gsap.com/)
- **Icons:** [Lucide React](https://lucide.dev/)
- **Language:** TypeScript

## Getting Started

### Prerequisites
- Node.js (v18 or higher recommended)
- npm or yarn

### Installation

1. Clone the repository:
   ```bash
   git clone https://github.com/amey-bauchkar/foremark-preview.git
   cd foremark-preview
   ```

2. Install dependencies:
   ```bash
   npm install
   ```

3. Start the development server:
   ```bash
   npm run dev
   ```

4. Open your browser and visit the local host address provided by Vite (usually `http://localhost:5173`) to view the application.

## Scripts

- `npm run dev` - Starts the Vite development server
- `npm run build` - Compiles TypeScript and builds the app for production
- `npm run preview` - Locally previews the production build
- `npm run lint` - Runs ESLint to check for code issues
